Highlighted Project

Final Year Project: SocialScope

Live Demo: socialscope.onrender.com

GitHub Repo: github.com/wtcov04/SocialScope

Video Demo: Watch on YouTube

SocialScope is a full-stack web application that allows users to input a TikTok or Instagram username and receive real-time analytics pulled from third-party APIs. It visualises profile metrics, post performance, engagement rates, and hashtag trends. Features include PDF report generation, Firebase login, and a benchmarking tool comparing user data to industry averages.
