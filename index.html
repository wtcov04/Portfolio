Featured Project: SocialScope

SocialScope – Research-Focused Social Media Analytics Tool

Live Demo: socialscope.onrender.com

GitHub: github.com/wtcov04/SocialScope

Video Demo: YouTube

Description: SocialScope is a web-based analytics platform designed to provide TikTok and Instagram insights without needing official API access or a business account. Created as a research-driven alternative to enterprise tools like Hootsuite and Meta Business Suite, it helps influencers and casual users analyse performance using public data.

Core Features:

  • Firebase Authentication: Secure login, session management, and profile saving functionality.
  • Profile Search: Retrieve real-time metrics like follower count, engagement rate, and bio from TikTok and Instagram using third-party APIs.
  • Visual Analytics: Interactive Chart.js graphs with support for bar, pie, and line charts. Users can customise views and metrics.
  • Hashtag Trends: PyTrends integration enables real-time tracking of hashtag popularity over time.
  • Benchmarking: Users can compare their analytics with normalised industry averages (e.g., likes, growth, engagement).
  • PDF Reports: Analytics pages can be exported to professional-quality PDF reports using WeasyPrint, embedding real-time charts and metrics.

Evaluation & Testing: The platform was tested with 20 users (9 influencers, 11 general users). Influencers appreciated the PDF export and saved profile features, while casual users praised its simplicity and chart interactivity. Survey insights and structured testing informed iterative UI and performance improvements.

Technologies: Flask (backend), Chart.js, Firebase (auth & DB), HTML/CSS/JS, RapidAPI (Instagram + TikTok), WeasyPrint, PyTrends
